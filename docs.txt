1. Git clone source code , di chuyển vô đúng thư mục và nhánh
2. Tạo venv từ file environment.yml , kích hoạt venv . nếu không được thì tạo venv , cài lib thủ công như trong file handmade.yml
   Cài thêm các thư viện sau :
    pip uninstall bitsandbytes -y
    pip install bitsandbytes==0.43.1
    pip install h5py
    pip install flash-attn --no-build-isolation
    pip uninstall deepspeed -y
    pip install deepspeed==0.15.4
    pip install --upgrade transformers==4.51.1
    pip install scikit-learn==1.6.1
    pip install nltk==3.9.1
    pip install rouge_score==0.1.2
    pip install dotenv==0.9.9
    pip install bert_score==0.3.13
    pip install FlagEmbedding==1.3.4
    pip install autotune==0.0.3

3. Sync Data dùng để huấn luyện chỉ cần file .jsonl là được. 
co the chay file download_data.sh de tai data. bash download_data.sh
4. Sửa đường dẫn tới file deepspeed config (ds_config.json) trong file train_base.py. Có thể sửa các siêu tham số 
cho quá trình huấn luyện trong file train_base.py ở hàm GRPOConfig.
5. Chạy lệnh bên dưới , nhớ chỉnh sửa đường dẫn tới file data .jsonl và tên model muốn train.
torchrun --nproc_per_node=2 RL_GRPO_test/train_base.py \
--dataset-path jsonl_TrainValMerge_noDescPerTurn_noDescCurr_onlyUtt_RL_max_emotion_max_strategy/test.jsonl --model-id 'Qwen/Qwen2-0.5B-Instruct'
